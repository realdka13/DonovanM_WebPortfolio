Model Rocket Flight Computer

**WORK IN PROGRESS**

The model rocket flight computer is a project to design and fabricate a computer to collect accelerometer, gyroscope, magnetometer, and altitude data and save it for later analysis. Later the computer will also be able to send the data wirelessly to a computer on the ground for telemetry. Additionally, the computer should be able to receive commands from the ground to do things like enable or disable data collection and transmission.

Details

8-Bit Breadboard Computer

The breadboard computer is nearly a copy of Ben Eaters 8-Bit Computer project designed to teach the basics of a Central Processing Unit. The computer is capable of addition and subtraction, has inturrupts, and has 16 bytes of RAM to use between variables and the program. It has 10 (11 if counting "No Operation") 4-bit instructions and an 8-bit bus to connect all of the modules. The highlight of this project is that is does not include a microcontroller, which is important to be able to understand how computers work at the most fundamental level.

Details

555 Square Wave Oscillator

This PCB was made for practicing PCB design and to learn reflow soldering. It takes in at least an 8V supply which it will regulate to 5V. The output is a 5V adjustable square wave using 555 timers, one in an astable, and other in a monostable configuration.
